Mengalami Error "Invalid Parameter" pada Browser Chrome?

Bila muncul kotak dengan judul "Invalid Parameter" atau "Parameter Tidak Sah", coba lakukan langkah-langkah di bawah ini.

  1. Matikan fitur Auto Translate di Chrome.
  2. Clear cache browser.
  3. Restart HP / laptop.
  4. Usahakan akses internet lancar.
